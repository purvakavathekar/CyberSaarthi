# CyberSaarthi (SafeGuard Academy)

**CyberSaarthi** is a mobile application designed to empower users with digital literacy and protection against online fraud. 

### Key Features:
* **Interactive Lessons:** Learn about UPI safety, phishing, and password security.
* **Practice Mode:** Simulated environments to identify scam calls and SMS.
* **AI Assistant:** Real-time help for detecting deepfakes and fraudulent content.
* **Multilingual Support:** Available in English, Hindi, Marathi, and more.
